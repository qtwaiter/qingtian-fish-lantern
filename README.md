# 青田鱼灯舞数字图谱

一个展示国家级非物质文化遗产——青田鱼灯舞的现代化数字展示平台。本项目基于郭秉强先生的手记整理制作，通过四季阵图体系呈现这一流传千年的民间艺术瑰宝。

## 项目简介

青田鱼灯舞是浙江省青田县的传统舞蹈，作为全球重要农业文化遗产"稻鱼共生"系统的艺术再现，深藏着明代开国元勋刘基（刘伯温）的军事智慧。本项目旨在通过数字化手段，将这一非物质文化遗产进行抢救性保护与传承。

### 核心特色

- **四季阵图体系**：完整展示春、夏、秋、冬及高潮部分"跳龙门"的阵法体系
- **双重文化内涵**：每个阵法都蕴含军事寓意与文化内涵
- **沉浸式体验**：使用现代化前端技术打造流畅的视觉体验
- **响应式设计**：完美适配桌面与移动设备

## 快速开始

### 环境要求

- Node.js >= 18.0.0
- pnpm >= 8.0.0

### 安装依赖

```bash
# 使用 pnpm 安装依赖
pnpm install
```

### 开发模式

```bash
# 启动开发服务器
pnpm dev
```

访问 http://localhost:5173 查看应用

### 生产构建

```bash
# 构建生产版本
pnpm build

# 启动生产服务器
pnpm start
```

生产服务器将运行在 http://localhost:3000

### 其他命令

```bash
# 类型检查
pnpm check

# 代码格式化
pnpm format

# 预览生产构建
pnpm preview
```

## 阵图体系

本项目展示了青田鱼灯舞的完整阵图体系，分为四个季节及高潮部分：

### 春（春鱼戏水）

1. **进门阵** - 鱼贯而入，神龙见首
   - 军事寓意：行军队列，井然有序
   - 文化内涵：象征万物复苏，生机勃勃

2. **春鱼戏水** - 泛游碧波，自由自在
   - 军事寓意：游击战术，灵活多变
   - 文化内涵：体现道家逍遥自在的生活哲学

### 夏（夏鱼跳滩）

1. **夏鱼跳滩** - 激流勇进，力争上游
   - 军事寓意：强攻险要，勇往直前
   - 文化内涵：象征不畏艰难，拼搏进取的精神

2. **四面八方阵** - 眼观六路，耳听八方
   - 军事寓意：全方位防御，无死角监控
   - 文化内涵：寓意行事周全，面面俱到

### 秋（秋鱼显籽）

1. **秋鱼显籽** - 金秋丰收，人丁兴旺
   - 军事寓意：侦察与反侦察，诱敌深入
   - 文化内涵：祈愿五谷丰登，子孙满堂

2. **鲤鱼抢珠** - 民族自强，奋力拼搏
   - 军事寓意：争夺战略要点，寸土必争
   - 文化内涵：只有努力拼搏，民族才能强大

### 冬（冬鱼结龙）

1. **冬鱼结龙** - 抱团御寒，众志成城
   - 军事寓意：收缩防线，集中兵力，瓮中捉鳖
   - 文化内涵：象征团结一心，共御外敌

### 高潮（跳龙门）

1. **跳龙门** - 一跃化龙，光耀门楣
   - 军事寓意：决战时刻，毕其功于一役
   - 文化内涵：激励族人发奋图强，改变命运

## 设计理念

### 视觉风格

项目采用**新中式水墨风格**，融合传统美学与现代设计语言：

- **配色方案**：采用米色（#F9F7F2）作为主背景色，营造温润古朴的氛围
- **水墨元素**：使用水墨笔触、印章、竖排文字等传统元素
- **留白美学**：借鉴中国传统绘画的留白技法，营造意境

### 交互体验

- **流畅动画**：使用 Framer Motion 实现流畅的页面过渡和元素动画
- **响应式布局**：完美适配各种屏幕尺寸
- **主题切换**：支持明暗主题切换（默认为浅色主题）

## 文化价值

青田鱼灯舞不仅是舞蹈艺术，更是：

1. **军事智慧的载体**：蕴含明代军事阵法思想
2. **稻鱼文化的再现**：体现全球重要农业文化遗产
3. **民族精神的象征**：体现团结、拼搏、自强不息的精神
4. **活态传承的典范**：通过数字化手段进行抢救性保护
